4. Key Concepts

  • Semantic HTML: Use <nav>, <ul>, <li> for accessibility
  • CSS Methods: Inline, Float, Flexbox each have trade-offs
  • Positioning: Dropdowns use relative + absolute positioning
  • Hover States: Provide visual feedback for better UX
  • Active States: Show current page location
