# webapp-demos

A collection of web development demo code for teaching — covering HTML/CSS, JavaScript, Node.js, Express, React, and TypeScript.

## Structure

| Folder | Topic |
|---|---|
| `html/`, `css/` | HTML & CSS fundamentals |
| `javascript/` | JavaScript basics |
| `nodejs/` | Node.js demos |
| `express/` | Express.js server examples |
| `react/`, `react-state/`, `react-effects/`, `react-conditional/`, `react-inter/` | React concepts (state, effects, conditional rendering, component interaction) |
| `typescript/` | TypeScript demos |
| `imgs-manipulation/` | Image manipulation examples |
| `refactor/` | Code refactoring examples |
| `docker/` | Docker setup examples |
| `my-app/`, `my-calculator/` | Sample applications |

## Purpose

These are teaching demos used in Computer Engineering courses at Khon Kaen University (KKU), covering web/mobile app development and programming fundamentals.

## Usage

Each folder is self-contained. Navigate into a folder and follow its own setup instructions (if any) or open the relevant HTML/JS file directly.
